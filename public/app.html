Setup guide

1

Copy your webhook URL

Copy the unique URL below and keep it handy.

2

Add it to each popup you want to track

In SendPulse, open a popup → Settings → Integrations → Webhooks → Add webhook URL. Paste the URL and enable "Send lead creation data". Repeat for every popup you want notifications for.

3

Popups appear automatically

Each popup shows up in the Popup routing section the first time a lead arrives from it. You don't need to configure them in advance.

4

Route popups to different channels (optional)

By default all leads go to one channel. Connect additional Slack channels using "+ Connect another channel" in the routing section, then assign any popup to any channel from the dropdown next to it.

5

Enable or disable individual popups

Use the toggle next to each popup to control whether its leads are forwarded. When any popup is enabled, only enabled popups send notifications — the rest are silently ignored.
